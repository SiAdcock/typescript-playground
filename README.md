# TypeScript Playground

## How to get there

This project requires [Gulp](http://gulpjs.com/) to be installed globally on your machine.

```
$ npm install gulp -g
$ npm install
```

## How to play

Start a node server:

```
$ npm start
```

Mess about with `index.ts`

### Build

```
$ npm run build
```

### Watch

```
$ gulp watch
```
